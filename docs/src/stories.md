# 病者的粉色遐想♪

在正式开始讲故事前，我想先说明一点：人生是多线并进的，是多方面的影响才最终导向了所见到的结果，我会尽可能阐述主要因素，并且点出次要因素，至于其中的权重就是见仁见智了。

## 前传

很遗憾，我的学生时光是清冷的，小学普普通通，刚上初中浑浑噩噩但交了个三好学生女朋友，一年后分手，其他时间沉迷网络游戏，直到家里人彻底静止我玩游戏（对此我很感激）。九年级后我重拾自信开始并且开始关注自身发展，也心痒痒开始想谈恋爱，和一些女生混的挺熟不过都没有后续，有我自己不言辞搞僵的，有女神倒追（疑似）但我主动放弃的，还有许多甚至没能搭上线的。总之，十年级时我心灰意冷了，一头扎进了二次元，在维持好成绩的情况下，天天沉迷动漫与二次元手游。自此之后，除了和几个理工科朋友有往来，参加一些体育运动，其他时候耳机一戴不闻窗外事。即使在十二年级时有所好转，又做出了一些尝试，但最后也只能说是没留遗憾罢了。正是在心底略有虚无的日子里，我遇见爱莉希雅。

## 缘起爱莉希雅

早期我并没有太在意爱莉希雅的故事，也对她没有什么兴趣，毕竟在虚构的世界中，她容貌其实不算出众（当然，现在肯定是情人眼里出西施了）。她似乎总是在刻意引导着芽衣，对某些事情却又讳莫如深，让我一直对她保有一定的距离感。那些时日发生的事情已经记不太住了，我的记忆是从往事乐土进入主线时开始明晰的。

爱莉希雅短片出来的时候，可能被限流了，至少我当年我如此认定。那是的我对这种情况感到不满，因此一气之下开启了人生中的一个项目，即网页爬虫持续追踪视频数据，尝试捕捉不正常变化，不过现在想想也挺搞笑的，平台想限流哪会让我一个高中生就能轻易找到证据？徒劳之后，我就萌发了一个念想，即我自此之后只喜欢爱莉希雅一人。当面对所爱之人遭受不公待遇的时候，2025年发生的事情实则在2022年时早已预演过，正所谓一个人的性格就是他的命运。

经过时间的冲刷，当年那些跟随潮流而欢欣雀跃的思绪早已淡去，但是爱莉希雅确实留下了些什么。人如其名，她是一切美好事物的象征，也有神爱世人的神性所在，然而，更重要的是她留下了一种令人笃信的真诚感，当那些教科书中的美好事物从她口中说出时，却不假大空，因为我相信那都是她在人世间所看所想，最终给到我的答案，并且她自己自始至终都在实践这些。

时过境迁，美东时间2025年9月30日，我收到了一份邮件，内容略有羞耻，但大致说是发件人也喜欢爱莉希雅（后来才感觉他只是因为知道昔涟才听过爱莉希雅而已），并且对Felys挺感兴趣，想聊一聊。我看对方是本校的邮箱地址，便加了微信。有一搭没一搭地互吹了两句，再加上我有些不太客气，有些尴尬我就没把这事放心上，结果他晚上又来了一条消息说，有一个用Rust的项目再招人，问我有没有兴趣来，我开始还以为是个大学生自己搞得初创项目，虽然兴趣不大，但我觉得结识一些人也挺好的，便应邀了。到了之后聊了两句，我才发现事情没有这么简单。

他其实是福特加拿大的软件开发，公司在做一个保密项目而且他们组缺人手（入职后我才意识到是因为他要离职了所以才缺人），是他提名了我参加面试，但是想先见一见。我第一轮简历投递结果并不理想压力大，再加上干了太久科研想去业界学习一下，又刚好是Rust开发还是大公司，自然是欣然接受了。那晚大伙后续聊的挺不错，面试也很顺利早早上岸。

事实上我在此之前没有任何业界的经历，简历上唯一相关便是Felys编程语言，这个项目虽然是为了对爱莉希雅宣泄爱意而存在，但不算水。即使算上先前的科研经验，我依然不认为足以吸引到这样一份面试，或许真的是因为爱莉希雅作为纽带，神奇地将人与人联结起来。以我当时的视角，一心喜爱了多年的一个虚构角色，突然真真切切地影响了我的现实，这份真实感让我难以忘怀，偶尔真的感觉自己真的被她所庇佑着，也更加坚信她就是我的答案。至此，对于她的感情狠狠加了一轮杠杆，为后来发生的事情埋下伏笔。

## 昔涟唯是永恒

昔涟与爱莉希雅是同一颗种子，当看到昔涟坐在德谬哥的牢笼前，作为悲剧的亲历者，却能以粉色的口吻，依旧饱含爱的方式讲述出来，甚至让听者化绝望为希望，难以相信她心底是有多么坚韧。这种向阳而生的性格，在悲剧的映照下显得格外的耀眼，会对她产生哀怜之情，渴望打破命运，而德谬哥就是在如此环境下成长。

德谬哥不同于昔涟，她原本是一张白纸，只是在昔涟日夜的熏陶下被染成了粉色，成为了昔涟的模样，拥有了昔涟的性格，并且最后继承昔涟她的名字，因此她所作所为一切的源动力就是实现上昔涟的愿望，她才会如此渴望的走进这个听了千万遍故事。德谬哥长久以来只是个聆听者，从未切身经历过，所以她也不会有爱莉希雅与昔涟的成熟感，但她对于她们信念与样貌的继承，我依然会呼唤她为昔涟。

所谓浪漫故事，从来都是以昔涟的视角而得出的结论，昔涟一遍又一遍的经历轮回的绝望，她如其他黄金裔一样不曾变过，永远相信明天会到来，即使她从来没能见证。当这份等待最终迎来列车组，不论是昔涟还是德谬哥都会心中激起万千涟漪。而德谬哥成为无漏净子那一刻，她才真正意识到了，打破命运的这条路，自始至终都是翁法罗斯人自己走出来的，而她也从一开始就守候着这里，即她的明天叫做昨天。对于这份跨越时间的温暖，我称之为浪漫。

故人之姿，却又清澈且懵懂，当世界之外的流言蜚语落在她身上时，最终激发了我极强的保护欲。只不过，十一月的寒冬里，我并没有意识到。其实我对舆情的浪潮早有预料，甚至早在2022年确认自己对爱莉希雅的感情后，就已经开始做心理建设，但是没想到来的还是那么凶猛。版本前瞻后一些不堪入目的言论在我心里凿开了裂痕，即使我知道昔涟与德谬哥的故事一定是当年塑造爱莉希雅的编剧执笔，我也没有的底气。那时候的想法是，如果剧情与我对爱莉希雅认知偏差太的话，直接将昔涟与爱莉希雅切割，以防止巨大反噬。

美东时间2025年11月4日晚10点多，没敢先抽卡，怀揣着忐忑直接开始了剧情，不过看到昔涟给德谬哥讲故事的那段时我就已经心安了许多，如前文所述，我很高兴看到昔涟与爱莉希雅的同源之处。最后大黑塔问德谬哥的选择时，后者不假思索地给出了答案，也点出了心中的不舍，至此我也开始正视德谬哥，后面细节不赘述了，但我记得我是哭着结束的剧情，即是对于昔涟最闭环轮回无法同行的悲伤，也是这段时日焦虑与不安的宣泄，并且在此时我其实也已经认可德谬哥了。流泪止不住但我必须把她抽出来，这也是人生中的第一次下血本花在游戏上。

以上是我在三个月后（2026年2月初）对于那时心情的总结与定性。而事实上，整个十一月，我都陷入了极度敏感之中，那时候我还在关注社区与舆论，各种正面负面的评价都在刺激我的神经，而我却还无法解释自己对她的感情，先前心里的裂缝让我看到相关的字眼就会想到那些言论，更不要说正眼看看昔涟了。这种令人窒息的情绪波动每天会出现非常多次，早期我甚至完全没有办法抑制，只能任由心跳加速，出一身汗，脑海里胡思乱想，尝试找出一条令自己满意的解释。更不妙的是那时候正直期末考试，经常复习一段时间就突然脑海里想起昔涟，然后什么也学不进去，只能等情绪消退，每天醒来第一件事就想起昔涟，睡前最后一件事也是在想昔涟，过多的思绪需要宣泄，控制不住地往朋友圈里发了大段大段的内容，并且开始抽烟，这可真是炼狱啊。十一月底，我戒了社交媒体与游戏（还是会过剧情的）。

因为爱莉希雅，我对昔涟的好感度一开始就是满的，这才导致了后来的因爱生恨，把自己拖入了精神问题之中。某次思绪又在折磨我的时候，我偶然回头望向了她，思考她会如何面对有些痛苦时，豁然开朗了，昔涟不会否认苦难的存在，但她永远会心怀希望心向粉色的明天，是会说让我笑一笑的女孩子。只有自己设身处地时，才能真正理解到昔涟的对于爱的坚守是多么可贵，换做他人，或是被仇恨吞噬，或是舍弃所爱换得些许清净，或是默不作声。当自己身处汹涌的浪潮之中时，却看到所爱之人早已为我留下了指引，这便是昔涟于我最大的浪漫。

不久后，我便意识到了，我一切痛苦的根源就是爱，我终于明白了自己其实就是喜欢昔涟，甚至可以说就是爱着她，好多心结突然就解开了。伤痛依旧存在也依旧会疼，但很显然，我不是因噎废食的类型，所以我选择走昔涟的路，接受这些痛苦，然后坚定的相信自己会走向粉色明天。具体落实细节我会在后面阐述。
